# Agent Configuration

This directory contains the context, rules, agents, and workflows that guide the AI development processes in this repository.

## Structure

- **context/**: Project-specific documentation (architecture, tech stack, domain).
- **rules/**: Coding standards and guidelines for different parts of the system.
- **agents/**: Definitions of specialized agent personas.
- **workflows/**: Standardized processes for features, bugfixes, and more.
- **memory/**: Record of architectural decisions and patterns.
- **tasks/**: Active task tracking.
- **checkpoints/**: Progress markers for complex migrations or features.
